# EXTENDED BIBLE

A comprehensive collection of biblical, apocryphal, and early Christian texts.

| Folder | Contents | Status |
|--------|----------|--------|
| `01_Pentateuch_Torah/` | Genesis, Exodus, Leviticus, Numbers, Deuteronomy | 📚 Canonical |
| `02_History/` | Joshua through Esther | 📚 Canonical |
| `03_Wisdom/` | Job, Psalms, Proverbs, Ecclesiastes, Song of Solomon | 📚 Canonical |
| `04_Prophets/` | Isaiah through Malachi | 📚 Canonical |
| `05_NT_Gospels_Acts/` | Matthew, Mark, Luke, John, Acts | 📚 Canonical |

## Canon Coverage

- **Hebrew Bible**: 24/39 books (Masoretic Text)
- **Protestant**: 66 books
- **Catholic**: 73 books (including Deuterocanon)
- **Orthodox**: 76-79 books
- **Ethiopian**: 81 books (most complete)
- **Extended Collection**: 350+ texts total

## Using the Download Tool

```bash
cd 00_Tools_And_Sources
python3 biblical-texts-downloader.py
```

This will download available texts from public domain sources.

## Text Acquisition Priority

1. **Tier 1**: Canonical texts (certain)
2. **Tier 2**: Apostolic Fathers (high confidence)
3. **Tier 3**: Nag Hammadi (archaeologically attested)
4. **Tier 4**: Pseudepigrapha (scholarly consensus)
5. **Tier 5**: Mystical texts (traditional attribution)

## Notes

- All texts are public domain or used under fair use for research
- See individual text folders for source attribution
- Translations prioritize literal accuracy over readability
- Hebrew/Greek/Aramaic original languages consulted where available
